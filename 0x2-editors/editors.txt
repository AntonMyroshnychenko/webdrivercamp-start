0 - open file Emacs:  emacs filename
1 - save file:   Ctrl + x Ctrl + s    or     Ctrl + x Ctrl + w
2 - cut an entire line:   C-k
3 - command to paste:   C-y
5 - command to undo:   C-x u      or      C-/
6 - command to quit:   C-x c
7 - command to insert text before the cursor:  i
    command to insert text after the cursor:   a
8 - command to delete and cut the current line:  dd
10 - command to undo what you just did:   u
11 - command to quit without saving:   :q!
12 - command to move the cursor to the start of the current line:   0 (zero)
13 - command to move the cursor to the end of the line:    $    or    A
